# vNERCAP ELT Simulator

**ELT Simulation Tool for MSFS SAR Training**

## Purpose
The **vNERCAP ELT Simulator** is a companion utility for **Microsoft Flight Simulator 2020, 2024, X-Plane 11, 12, and Prepar3D V5, V6.** that simulates **Emergency Locator Transmitter (ELT)** signals for virtual CAP-style Search and Rescue (SAR) missions.

## Scope
- Virtual simulation and training only  
- No real radio transmissions or emergency response  
- Not affiliated with Civil Air Patrol or the USAF Auxiliary  

## Features
- Simulated ELT signal source  
- Relative bearing and signal strength output  
- Supports standard SAR search patterns  

## Requirements
- A Flight Simulator (Listed Above) 
- Windows 10/11  

## Use
1. Launch MSFS and position aircraft per mission brief  
2. Start the ELT Simulator  
3. Conduct ELT search and report results  

## Compliance
Use must comply with VATSIM policy and **vNERCAP 1-1**.

## Disclaimer
For training and entertainment only. Not for real-world SAR use.
